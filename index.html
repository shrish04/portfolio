Solar Power Prediction using GA & PSO Techniques of Machine Learning

Published in: Renewable Energy Journal

View Paper November 2025

This research focuses on predicting solar power generation more accurately using machine learning. It combines two powerful techniques — Genetic Algorithm (GA) and Particle Swarm Optimization (PSO) — to improve how models like LSTM, Random Forest, and SVR learn from data. The system studies past weather information such as sunlight, temperature, and humidity to predict how much solar energy will be produced. By using GA and PSO, the model becomes more accurate, faster, and reliable, helping in better use of renewable energy and efficient power management.

Forest Fire Detection System Using Wireless Sensor Networks (WSN)

Presented at: International Sensor Systems Conference (ISSC)

View Paper

This research focuses on creating a smart system to detect forest fires early using Wireless Sensor Networks (WSN) and machine learning. The system uses small sensors that check temperature, humidity, smoke, and gas levels in forests. These sensors send real-time data to a base station, where a machine learning model analyzes it to quickly identify any fire risk. The system is designed to be low-cost, energy-efficient, and highly accurate, making it useful for preventing large-scale wildfires and protecting the environment.
